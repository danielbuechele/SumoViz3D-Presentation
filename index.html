Webbasiert

Webbasiert

  • Plattformunabhängig
  • keine Installation notwendig
  • gemeinsame Datennutzung
  • einfache Weitergabe
  • einfaches Ausrollen von Updates







<canvas></canvas>

canvas

  • frei nutzbare Zeichenfläche
  • 2D- oder 3D-Inhalte
  • Zeichnen via JavaScript
  • nicht bearbeitbar

3D

  • basierend auf OpenGL ES 2.0
  • hardwarebeschleunigt
  • 3D-Grafik im canvas-Element
  • Bibliotheken mit Standard-Elementen

Rendering

  1. Anlegen der 3D-Modelle

  3. Rasterisierung
  4. Fragment-Shader
  5. Darstellung im canvas-Element
